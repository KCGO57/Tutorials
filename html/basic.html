HTML Basics

HTML is a markup language, which provides us with a way in which we can create a structure for our webpage!
We always start our document with <!DOCTYPE html>. What this does is declaring the document type, heloing the browser display web pages correctly.
To add a heading to our website, we will use the <h1>-<h6> tags.
h1 will be the biggest header, while h6 will be the smallest header

This is H1

This is H6
You will place your desired text within the brackets like this:
<h1>Text goes here</h1>
Be mindful that every closing tag will have a backwards slash (\) before the name of the tag

Now lets discuss paragraphs. In order to start a paragraph, you will need to use the <p> tag.
Links and images are also important pieces of information to know.
A link is defined with the <a> tag
This is used a little bit differentyl than the previous tags, because will will need to add an attribute to make it work correctly.
Attributes are used to add additional information to HTML elements
An example:
<a href = "link of choice"> Title of link</a>
The href attribute specifies the links destination.

Now, lets go over images which are also written a bit differenty that the previous tags.
Images use the <img> tag and DO NOT have a closing tag.
It uses the attribute src which specifies the source files, alt: which is the alternative text, and we can also give it the attribute width and height to make it as big or small as we please
Here is an example of what it would look like:
<img src = "images link" alt = "alternative text" width = "50" height = 50>

Now, lets put all of this together and make our first website!
<!DOCTYPE html>
<html>
<body>

<h1>My First Website</h1>
<p>Welcome to my first website. Enjoy your stay!</p>
Here is a link to my favorite song: <a href="https://www.w3schools.com">This is a link</a>
Here is the singer of that song: <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
