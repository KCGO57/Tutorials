Styles

In the previous lesson, we went over attributes. In this one, we weill focus on the style attribute
The style attribute is used to well style each element. Changing things such as the size, color, or the font!
Here is the following syntax:
<tagname style = "property:value;">
Both property and value in this case will both be a CSS property or value-- which we will learn in later lessons.
To set the background color of an HTML element, we will use the background-color property. The value will then be our chosen color.
Example: <h3 style = "background-color: pink;">Example</h3>
